# CIS 486 Student CRUD MVC App

A full stack MVC-style student CRUD application built with Node.js, Express, MongoDB, Bootstrap 5, jQuery, and normalize.css.

## Author
Alexander Nunley

## Features
- User registration and login
- Create, read, update, and delete students
- MongoDB persistent storage
- RESTful API
- Bootstrap 5 and jQuery front end
- GitHub Actions CI

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- dotenv
- nodemon
- Bootstrap 5
- jQuery
- normalize.css

## Local Setup
1. Clone repo
2. Run `npm install`
3. Create `.env`
4. Run `npm run dev`

## Environment Variables
- PORT
- MONGO_URI
- JWT_SECRET

## Deployment
- Dev: Render
- Production: GCP

## Branch Workflow
- `main` for stable code
- `dev` for development
- feature branches and pull requests for changes
